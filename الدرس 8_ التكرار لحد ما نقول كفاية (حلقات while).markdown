"يا شباب، تخيلوا إنكم بتلعبوا لعبة وبتقولوا: 'هكمل ألعب لحد ما أتعب'. في البرمجة، فيه حاجة زي كده اسمها حلقة `while`! يعني الكمبيوتر هيفضل يكرر لحد ما نقوله: 'كفاية كده'. النهاردة هنعمل برامج تكرر بناءً على شرط، يلا بينا!"
**5-20 دقيقة: شرح حلقات `while`**
"الحلقة `while` بتكرر الأوامر طالما شرط معين صح. يعني زي إنك بتقول: 'طالما عندي فلوس، هاشتري أيس كريم'. خلونا نجرب مثال: لو عايزين نطبع الأرقام من 1 لـ5:
```python
number = 1
while number <= 5:
    print(number)
    number = number + 1
```
لما نشغل الكود، هيطبع:
1
2
3
4
5
خلونا نفهم الكود خطوة بخطوة:
- `number = 1` يعني بنبدأ بصندوق فيه 1.
- `while number <= 5:` يعني كرر طالما `number` أقل من أو يساوي 5.
- `print(number)` بيطبع قيمة `number`.
- `number = number + 1` يعني زوّد `number` بواحد في كل مرة، عشان نوصل لـ5 ونوقف.
- لو نسينا `number = number + 1`، الحلقة هتفضل تكرر للأبد، وده غلط كبير!
- المسافات (4 مسافات) قبل `print` و`number = number + 1` مهمة، عشان الكمبيوتر يعرف إن الأوامر دي تبع الحلقة.
نقدر نستخدم `while` لشروط تانية. مثلاً، لو عايزين نطبع 'أكل أيس كريم' طالما عندنا فلوس:
```python
money = 10
while money >= 2:
    print('Eat ice cream!')
    money = money - 2
```
هيطبع 'Eat ice cream!' 5 مرات، عشان الفلوس هتنقص من 10 لحد ما تبقى أقل من 2.
*(ارسم على السبورة خط زمني: 'number=1 → 2 → 3 → 4 → 5 → توقف'. وضح إزاي الشرط بيحدد التكرار.)*

##### **الهدف التعليمي**
- فهم حلقات `while` لتكرار الأوامر بناءً على شرط.
- تحديث المتغيرات داخل الحلقة لتجنب التكرار اللانهائي.
- كتابة برامج بسيطة تستخدم `while`.
##### **مثال عملي**
```python
# Number repetition
number = 1
while number <= 5:
    print(number)
    number = number + 1
# Repetition based on money
money = 10
while money >= 2:
    print('Eat ice cream!')
    money = money - 2
```
##### **النشاط التطبيقي**
- **المهمة 1**: كتابة برنامج يطبع الأرقام من 1 لـ7.
- **المهمة 2**: تجربة تكرار كلام أو أرقام مختلفة.
- **الكود**: كما في المثال العملي.
- **الطريقة**: كتابة الكود في Thonny أو Replit.
##### **التحدي الصغير**
- **السؤال**: اكتب كود يحسب عدد الحلوى اللي نقدر نشتريها بـ15 جنيه (كل حلوى 3 جنيه).
- **مثال إجابة**:
  ```python
  money = 15
  candies = 0
  while money >= 3:
      candies = candies + 1
      money = money - 3
  print('We can buy', candies, 'candies')
  ```
##### **نصايح للمعلم**
- وضح فكرة `while` بمثال زي 'أكل لحد ما أشبع'.
- لو طفل نسي تحديث المتغير (زي `number = number + 1`)، اشرح إن الحلقة هتفضل تكرر للأبد.
- شجعهم يجربوا شروط مختلفة، زي `money >= 5`.
- لو في وقت، خليهم يكتبوا برنامج يعد للخلف (من 10 لـ1).
- لو في أخطاء، زي نسيان المسافات، اشرح إنها زي تنسيق القصة.
