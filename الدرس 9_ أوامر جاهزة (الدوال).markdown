##### **سكريبت الشرح (بلهجة مصرية) - 60 دقيقة**
**0-5 دقايق: مقدمة ممتعة**
"يا شباب، تخيلوا إنكم عايزين تعلموا أخوكم الصغير يعمل ساندويتش كل يوم، هتشرحوله كل مرة؟ لا، هتعملوله زرار سحري تضغط عليه يعمل الساندويتش على طول! في البرمجة، الحاجة دي اسمها **الدوال**. النهاردة هنتعلم إزاي نعمل أوامر جاهزة نستخدمها في أي وقت، وهنخلي الكمبيوتر يعمل حاجات زي التحية أو الحساب بضغطة زرار. جاهزين؟ يلا!"
**5-20 دقيقة: شرح الدوال**
"الدالة زي أمر جاهز تكتبه مرة واحدة، وبعدين تستخدمه كتير من غير ما تكتب الكود من الأول. في بايثون، بنعمل دالة بكلمة `def`. خلونا نجرب مثال: لو عايزين نعمل دالة تقول مرحبا لأي اسم:
```python
def greet(name):
    print('Hello', name, '!')
greet('Ali')
greet('Sara')
```
لما نشغل الكود، هيطلع:
Hello Ali !
Hello Sara !
خلونا نفهم الكود خطوة بخطوة:
- `def greet(name):` يعني اعمل دالة اسمها `greet`، وخليها تاخد صندوق اسمه `name`.
- `name` ده زي متغير، بس بيشتغل بس جو الدالة. ممكن نسميه أي حاجة، زي `اسم` أو `الاسم`.
- `: ` و4 مسافات قبل `print` مهمين، عشان الكمبيوتر يعرف إن الأمر ده تبع الدالة.
- `print('Hello', name, '!')` هو الأمر اللي هيتنفذ لما نستدعي الدالة.
- `greet('Ali')` يعني شغل الدالة وحط 'Ali' في الصندوق `name`.
- نقدر نستدعي الدالة كتير زي ما عايزين، وكل مرة بنغير الاسم.
نقدر نعمل دالة تعمل حاجات تانية، زي الحساب. مثلاً، دالة تحسب مربع أي رقم:
```python
def square(number):
    result = number * number
    print('Square of', number, 'is', result)
square(3)
square(5)
```
هيطلع:
Square of 3 is 9
Square of 5 is 25
خلونا نفهم:
- `number` هو المتغير اللي بنحطه في الدالة.
- `result = number * number` يعني احسب مربع الرقم.
- `square(3)` يعني شغل الدالة وحط 3 في `number`.
*(ارسم على السبورة زرار مكتوب عليه 'greet'، وجنبه اسماء زي 'Ali' و'Sara'، ووضح إزاي الدالة بتشتغل زي الزرار.)*

```python
double(4)

```

**40-55 دقيقة: تحدي صغير + مناقشة**
"عندنا تحدي ممتع! هنعمل دالة تحسب تكلفة الحلوى بناءً على العدد. لو كل حلوى بـ3 جنيه، اكتبوا الكود كده:
```python
def candy_cost(count):
    cost = count * 3
    print('Cost of', count, 'candies is', cost, 'pounds')
candy_cost(5)
candy_cost(2)
```
خلونا نفهم:
- `count` هو عدد الحلوى اللي هنحسب تكلفتها.
- `cost = count * 3` يعني اضرب العدد في 3.
- `candy_cost(5)` يعني احسب تكلفة 5 حلوى.
جربوا، وغيروا العدد لـ10 أو 3. مين هيشارك تكلفته؟"

**55-60 دقيقة: تلخيص وتشجيع**
"برافو عليكم يا شباب! النهاردة اتعلمنا إزاي نعمل أوامر جاهزة (دوال) نستخدمها زي زرار سحري. المرة الجاية هنتعلم إزاي نحط حاجات كتير في قايمة زي قايمة التسوق. فكروا في أوامر عايزين تعملوها، زي دالة تحسب درجات أو تقول نكتة، وهنشوفها مع بعض!"
##### **الهدف التعليمي**
- فهم الدوال وكيفية إنشاؤها باستخدام `def`.
- استخدام المتغيرات داخل الدوال (Parameters).
- كتابة برامج بسيطة تستخدم دوال.
##### **مثال عملي**
```python
# Greeting function
def greet(name):
    print('Hello', name, '!')
greet('Ali')
greet('Sara')
# Calculation function
def square(number):
    result = number * number
    print('Square of', number, 'is', result)
square(3)
```
##### **النشاط التطبيقي**
- **المهمة 1**: كتابة دالة تقول مرحبا باسم الطفل.
- **المهمة 2**: كتابة دالة تحسب ضعف رقم.
- **الكود**: كما في المثال العملي.
- **الطريقة**: كتابة الكود في Thonny أو Replit.
##### **التحدي الصغير**
- **السؤال**: اكتب دالة تحسب تكلفة الحلوى (كل حلوى 3 جنيه).
- **مثال إجابة**:
  ```python
  def candy_cost(count):
      cost = count * 3
      print('Cost of', count, 'candies is', cost, 'pounds')
  candy_cost(5)
  ```
##### **نصايح للمعلم**
- وضح فكرة الدالة بمثال زي 'زرار يعمل عصير'.
- لو طفل نسي المسافات أو الأقواس، اشرح إنها زي تنسيق الوصفة.
- شجعهم يجربوا دوال بأسماء مضحكة، زي `joke(joke)`.
- لو في وقت، خليهم يكتبوا دالة تحسب شيء تاني، زي تكلفة ألعاب.
- لو في أخطاء، زي نسيان استدعاء الدالة، اشرح إن الدالة زي أمر لازم 'نشغله'.
