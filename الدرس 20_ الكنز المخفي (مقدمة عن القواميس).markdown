##### **سكريبت الشرح (بلهجة مصرية) - 60 دقيقة**
**0-5 دقايق: مقدمة ممتعة**
"يا شباب، تخيلوا إن عندكم صندوق كنز سحري، كل ما تكتبوا اسم حاجة، يطلعلكم معلومة عنها! زي إنكم تكتبوا 'بيتزا'، يقولكم 'أكلة إيطالي'. في البرمجة، الصندوق ده اسمه **قاموس** (Dictionary). النهاردة هنتعلم إزاي نستخدم القواميس عشان ننظم بيانات زي أسماء ودرجات، أو ألعاب وأسعار. جاهزين؟ يلا!"
**5-20 دقيقة: شرح القواميس**
"القاموس في بايثون زي دفتر تليفونات، كل اسم ليه رقم. في القاموس، عندنا **مفتاح** (Key) و**قيمة** (Value). المفتاح زي الاسم، والقيمة زي المعلومة اللي بنربطها بيه. بنعمل قاموس باستخدام الأقواس المتعرجة `{}`. خلونا نجرب مثال:
```python
grades = {'Ahmed': 90, 'Sarah': 85, 'Ali': 95}
print(grades)
print('Ahmed\'s grade:', grades['Ahmed'])
```
لما نشغل الكود، هيطلع:
{'Ahmed': 90, 'Sarah': 85, 'Ali': 95}
Ahmed's grade: 90
خلونا نفهم الكود:
- `grades` هو اسم القاموس.
- `'Ahmed': 90` يعني المفتاح هو 'Ahmed' والقيمة هي 90.
- الفواصل `,` بتفصل بين كل مفتاح وقيمته.
- `grades['Ahmed']` يعني جيب القيمة اللي مربوطة بمفتاح 'Ahmed'.
نقدر نضيف حاجة جديدة للقاموس:
```python
grades['Mona'] = 88
print(grades)
```
هيطلع: {'Ahmed': 90, 'Sarah': 85, 'Ali': 95, 'Mona': 88}.
نقدر كمان نغيّر قيمة موجودة:
```python
grades['Sarah'] = 87
print(grades)
```
هيطلع: {'Ahmed': 90, 'Sarah': 87, 'Ali': 95}.
ولو عايزين نطبع كل المفاتيح أو القيم:
```python
print('Names:', grades.keys())
print('Grades:', grades.values())
```
هيطلع:
Names: dict_keys(['Ahmed', 'Sarah', 'Ali'])
Grades: dict_values([90, 87, 95])
نقدر نكرر على القاموس باستخدام حلقة `for`:
```python
for name in grades:
    print(name, 'scored', grades[name])
```
هيطلع:
Ahmed scored 90
Sarah scored 87
Ali scored 95
*(ارسم على السبورة صندوق مكتوب عليه 'قاموس'، وجواه أسماء زي 'Ahmed: 90'، ووضح إزاي المفتاح بيفتح القيمة.)*

"برافو عليكم يا شباب! النهاردة اتعلمنا إزاي نستخدم القواميس زي صندوق كنز عشان ننظم بيانات. المرة الجاية هنعمل مشروع كبير يجمع كل اللي اتعلمناه، زي لعبة أو برنامج رسم. فكروا في قواميس عايزين تعملوها، زي قاموس أبطال خارقين، وهنشوفها مع بعض!"
##### **الهدف التعليمي**
- فهم القواميس وكيفية إنشاؤها باستخدام `{}`.
- استخدام المفاتيح والقيم للوصول إلى البيانات وتعديلها.
- كتابة برامج بسيطة تستخدم القواميس مع حلقات `for`.
##### **مثال عملي**
```python
# إنشاء قاموس
grades = {'Ahmed': 90, 'Sarah': 85, 'Ali': 95}
print(grades)
print('Ahmed\'s grade:', grades['Ahmed'])
# إضافة عنصر
grades['Mona'] = 88
print(grades)
# تكرار القاموس
for name in grades:
    print(name, 'scored', grades[name])
```
##### **النشاط التطبيقي**
- **المهمة 1**: إنشاء قاموس لألعاب وأسعارها وطباعة سعر لعبة.
- **المهمة 2**: إضافة لعبة جديدة وطباعة كل الألعاب باستخدام حلقة.
- **الكود**: كما في النشاط.
- **الطريقة**: كتابة الكود في Thonny أو Replit.
##### **التحدي الصغير**
- **السؤال**: إنشاء قاموس لأكلات وأسعارها وطباعة رسالة لكل أكلة.
- **مثال إجابة**:
  ```python
  foods = {'Pizza': 50, 'Koshary': 20, 'Burger': 30}
  for food in foods:
      print('The food', food, 'costs', foods[food], 'EGP')
  ```
##### **نصايح للمعلم**
- وضح فكرة القاموس بمثال زي دفتر درجات المدرسة.
- لو طفل كتب مفتاح غلط (زي `'Ahmed'` بدل `'Ahmed'`)، اشرح إن المفتاح لازم يكون زي ما كتبناه بالظبط.
- شجعهم يجربوا قواميس مضحكة، زي قاموس أبطال خارقين ومهاراتهم.
- لو في وقت، خليهم يجربوا حذف مفتاح باستخدام `pop()` (بس اشرحها الأول).
- لو في أخطاء، زي نسيان `: ` بين المفتاح والقيمة، اشرح إنها زي رابط بين الاسم والمعلومة.
