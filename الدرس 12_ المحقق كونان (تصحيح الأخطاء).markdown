##### **سكريبت الشرح (بلهجة مصرية) - 60 دقيقة**
**0-5 دقايق: مقدمة ممتعة**
"يا شباب، تخيلوا إنكم محققين زي كونان، ومهمتكم تلاقوا الأخطاء في برنامج مش شغال! في البرمجة، الأخطاء زي اللغز، لازم نفكره عشان الكود يشتغل. النهاردة هنتعلم إزاي نلاقي الأخطاء (اللي بنسميها Bugs) ونصلحها زي المحترفين. جاهزين تصيروا محققين؟ يلا!"
**5-20 دقيقة: شرح تصحيح الأخطاء**
"لما تكتب كود ويطلع خطأ، الكمبيوتر بيقولك: 'فيه حاجة غلط!'، بس لازم نفهم إيه الغلط ده. فيه أنواع أخطاء كتير، زي:
- **أخطاء الكتابة (Syntax Errors)**: زي إنك تنسى علامة `:` أو تكتب كلمة غلط.
- **أخطاء المنطق (Logic Errors)**: الكود بيشتغل، بس النتيجة غلط، زي إنك تجمع بدل ما تطرح.
- **أخطاء التشغيل (Runtime Errors)**: زي إنك تحاول تقسم على صفر أو تستخدم نص كأنه رقم.
خلونا نجرب مثال لكود فيه خطأ:
</div>

<div dir="ltr">

```python

print('Hello', name  # missing quotation mark
```

<div dir="rtl">

</div>

<div dir="ltr">

```python

```

</div>

<div dir="rtl">

مثال تاني، كود بيحسب تكلفة حلوى بس فيه خطأ منطقي:
</div>

```python
print('The cost is', cost, 'pounds')
```

<div dir="rtl">

لو كتبت 5، هيطلع: The cost is 8 pounds، وده غلط عشان 5 حلوى المفروض 15 جنيه (5 × 3). الحل: نغير `+` لـ`*`:
</div>

<div dir="ltr">

```python
cost = candy_count * 3
```

نصايح عشان تصير محقق كويس:
1. اقرأ رسالة الخطأ اللي الكمبيوتر بيطلعها، هتقولك السطر اللي فيه المشكلة.
2. جرب الكود خطوة بخطوة، وشوف إيه اللي مش شغال.
3. استخدم `print` عشان تشوف قيم المتغيرات، زي: `print(candy_count)` قبل الحساب.
4. اسأل نفسك: 'النتيجة دي منطقية؟' لو لأ، ابحث عن الغلط.
*(ارسم على السبورة كود فيه خطأ، زي نسيان `:`، وجنبه علامة X، وبعدين الكود الصحيح بعلامة √. وضح إزاي المحقق بيحل اللغز.)*

##### **الهدف التعليمي**
- فهم أنواع الأخطاء (Syntax، Logic، Runtime).
- تعلم كيفية قراءة رسائل الخطأ وتصحيح الكود.
- تطوير مهارة حل المشاكل (Debugging).
##### **مثال عملي**
</div>

<div dir="ltr">

```python
# خطأ كتابة
name = input('What is your name?')
print('Hello', name, '!')  # كان فيه علامة ناقصة، صلحناها
# خطأ منطقي
candy_count = int(input('How many candies? '))
cost = candy_count * 3  # كان +، صلحناه لـ*
print('The cost is', cost, 'pounds')
```

</div>

<div dir="rtl">

##### **النشاط التطبيقي**
- **المهمة 1**: إيجاد خطأ في كود يسأل عن السن (نسيان `)`).
- **المهمة 2**: إيجاد خطأ منطقي في كود يحسب عدد الفاكهة (استخدام `-` بدل `+`).
- **الكود**: كما في النشاط.
- **الطريقة**: كتابة الكود وتصحيحه.
##### **التحدي الصغير**
- **السؤال**: إيجاد وتصحيح خطأ في كود يفحص الاسم (نسيان `:`).
- **مثال إجابة**:
  </div>

<div dir="ltr">

```python
  اسم = input('إيه اسمك؟ ')
  if اسم == 'أحمد':
      print('مرحبا أحمد!')
  else:
      print('مرحبا غريب!')
  ```

</div>

##### **نصايح للمعلم**
- وضح الأخطاء بمثال زي كتابة وصفة أكل غلط.
- لو طفل مش عارف يلاقي الخطأ، اقترح إنه يستخدم `print` لفحص المتغيرات.
- شجعهم يشاركوا أخطاء مضحكة، زي طباعة اسم غلط.
- لو في وقت، خليهم يجربوا كود فيه أكتر من خطأ ويحاولوا يصلحوه.
- لو في خطأ غريب، اشرح إن Debugging زي حل لغز، وكل خطأ له حل.
