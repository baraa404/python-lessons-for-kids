##### **سكريبت الشرح (بلهجة مصرية) - 60 دقيقة**
**0-5 دقايق: مقدمة ممتعة**
"يا شباب، جاهزين نخلي الكمبيوتر يفكر زينا؟ النهاردة هنتعلم حاجة جامدة اسمها **الشروط**! تخيلوا إنكم بتقولوا للكمبيوتر: 'لو الجو حر، افتح المكيف، لو برد، شغل الدفاية'. الكمبيوتر هيبقى زي صديق ذكي بياخد قرارات. هنعمل برامج تقرر حاجات بنفسها، يلا بينا!"
**5-20 دقيقة: شرح الشروط (`if-else`)**
"في بايثون، الشروط بتستخدم كلمة `if`، ودي معناها 'لو'. يعني لو حاجة صح، اعمل كده، ولو مش صح، اعمل حاجة تانية. خلونا نجرب مثال: لو عايزين نعرف لو السن بتاعنا يسمحلنا نركب الدراجة ولا لأ:
```python
age = 11
if age > 10:
    print('You can ride the bike!')
else:
    print("You're too young to ride the bike")
```
لما نشغل الكود، هيطلع: You can ride the bike!
خلونا نفهم الكود ده خطوة بخطوة:
- `age = 11` يعني حطينا 11 في صندوق اسمه `age`.
- `if age > 10:` يعني لو السن أكبر من 10، نفذ الأمر اللي تحت.
- `>` دي علامة تعني 'أكبر من'. فيه علامات تانية زي `<` (أصغر من)، `==` (يساوي)، `!=` (مش يساوي).
- `: ` دي زي إنك بتقول: 'ابدأ هنا'.
- `print('You can ride the bike!')` هيتكتب لو الشرط صح (يعني السن أكبر من 10).
- `else:` يعني 'وإلا'، يعني لو الشرط مش صح (يعني السن 10 أو أقل)، نفذ الأمر اللي تحت.
- المسافات (4 مسافات أو Tab) قبل `print` مهمة جدًا، عشان الكمبيوتر يعرف إن الأوامر دي تبع `if` أو `else`. لو نسيت المسافات، الكمبيوتر هيزعل ويطلع خطأ!
نقدر نجرب شرط تاني، زي لو عايزين نعرف الرقم زوجي ولا فردي:
```python
number = 4
if number % 2 == 0:
    print('The number is even')
else:
    print('The number is odd')
```
هيطلع: The number is even.
خلونا نفهم:
- `number % 2` يعني الباقي لما نقسم الرقم على 2. لو الرقم زوجي، الباقي هيبقى 0.
- `== 0` يعني بنسأل: هل الباقي يساوي 0؟
- لو صح، يطبع 'The number is even'، ولو مش صح، يطبع 'The number is odd'.
*(ارسم على السبورة شجرة قرارات: 'لو السن > 10، نعم: ركوب دراجة، لا: ممنوع'. وضح إزاي الكمبيوتر بيقارن الأرقام.)*

**55-60 دقيقة: تلخيص وتشجيع**
"برافو عليكم يا شباب! النهاردة اتعلمنا إزاي نخلي الكمبيوتر ياخد قرارات ذكية زي إنسان باستخدام `if` و`else`. المرة الجاية هنتعلم إزاي نكرر الحاجات بدل ما نكتبها كتير. فكروا في قرارات عايزين الكمبيوتر ياخدها، زي 'لو عندي فلوس، أشتري أيس كريم'، وهنشوفها مع بعض!"
##### **الهدف التعليمي**
- فهم الشروط (`if-else`) لاتخاذ قرارات.
- استخدام علامات المقارنة (`>`, `<`, `==`, `>=`, `!=`).
- كتابة برامج بسيطة تستخدم الشروط.
##### **مثال عملي**
```python
# فحص السن
age = 11
if age > 10:
    print('You can ride the bike!')
else:
    print("You're too young to ride the bike")
# فحص زوجي أو فردي
number = 4
if number % 2 == 0:
    print('The number is even')
else:
    print('The number is odd')
```
##### **النشاط التطبيقي**
- **المهمة**: كتابة برنامج يفحص إذا كان السن يسمح بركوب الدراجة، وتجربة أرقام مختلفة.
- **الكود**: كما في المثال العملي.
- **الطريقة**: كتابة الكود في Thonny أو Replit.
##### **التحدي الصغير**
- **السؤال**: اكتب كود يفحص إذا كانت درجة الامتحان ناجحة (50 أو أكتر) أو راسبة.
- **مثال إجابة**:
  ```python
  grade = 60
  if grade >= 50:
      print('Congratulations, you passed!')
  else:
      print('Better luck next time, you failed')
  ```
##### **نصايح للمعلم**
- ارسم شجرة قرارات على السبورة لتوضيح `if-else`.
- لو طفل نسي المسافات أو علامة `:`، اشرح إنها زي علامات الترقيم في الكلام.
- شجعهم يجربوا شروط مختلفة، زي فحص لو الرقم أكبر من 100.
- لو في وقت زيادة، خليهم يكتبوا شرط باستخدام `!=` (مش يساوي).
- لو في أخطاء، زي كتابة `=` بدل `==`، اشرح الفرق بمثال بسيط.
