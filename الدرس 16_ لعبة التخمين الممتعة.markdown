##### **سكريبت الشرح (بلهجة مصرية) - 60 دقيقة**
**0-5 دقايق: مقدمة ممتعة**
"يا شباب، جاهزين تلعبوا لعبة ممتعة؟ هنعمل لعبة تخمين، الكمبيوتر هيختار رقم سري، وإنتوا هتحاولوا تتخمنوه! لو تخمنتوا غلط، هيديكم تلميح زي 'الرقم أكبر' أو 'الرقم أصغر'. هنستخدم كل اللي اتعلمناه زي `input` والشروط وحلقات `while`. يلا، خلونا نلعب!"
**5-20 دقيقة: شرح لعبة التخمين**
"هنعمل برنامج يختار رقم سري بين 1 و10، ويسأل اللاعب يتخمن الرقم. لو التخمين غلط، يدي تلميح ويخلي اللاعب يجرب تاني. هنستخدم مكتبة `random` عشان نختار رقم عشوائي. الكود كده:
</div>

<div dir="ltr">

```python
import random
secret_number = random.randint(1, 10)
print('I chose a secret number between 1 and 10, try to guess it!')
while True:
    guess = int(input('What is your guess? '))
    if guess == secret_number:
        print('Great! You found the secret number!')
        break
    elif guess < secret_number:
        print('No, the secret number is bigger than', guess)
    else:
        print('No, the secret number is smaller than', guess)
```

لما نشغل الكود، هيسألك تتخمن الرقم، ويظل يديك تلميحات لحد ما تلاقي الرقم السري.
خلونا نفهم الكود:
- `import random` يجيب مكتبة الأرقام العشوائية.
- `random.randint(1, 10)` يختار رقم عشوائي بين 1 و10 (يعني 1، 2، ...، 10).
- `while True:` يعني كرر للأبد لحد ما نقوله يتوقف.
- `guess = int(input(...))` ياخد تخمين اللاعب ويحوله لـرقم.
- `if guess == secret_number:` لو التخمين صح، يطبع 'Great' ويوقف الحلقة بـ`break`.
- `elif guess < secret_number:` لو التخمين أصغر، يدي تلميح إن الرقم أكبر.
- `else:` لو التخمين أكبر، يدي تلميح إن الرقم أصغر.
*(ارسم على السبورة أرقام من 1 لـ10، وجنبها لاعب بيتخمن، وسهم يشير للرقم السري. وضح إزاي التلميحات بتساعد.)*

```python
import random
secret_number = random.randint(1, 10)
attempts = 0
print('I chose a secret number between 1 and 10, try to guess it!')
while True:
    guess = int(input('What is your guess? '))
    attempts = attempts + 1
    if guess == secret_number:
        print('Great! You found the secret number in', attempts, 'attempts!')
        break
    elif guess < secret_number:
        print('No, the secret number is bigger than', guess)
    else:
        print('No, the secret number is smaller than', guess)
```

##### **الهدف التعليمي**
- إنشاء لعبة تخمين باستخدام `random`, `input`, و`while`.
- فهم استخدام الأرقام العشوائية والتحكم في التكرار.
- تطبيق الشروط والحلقات في مشروع تفاعلي.
##### **مثال عملي**
