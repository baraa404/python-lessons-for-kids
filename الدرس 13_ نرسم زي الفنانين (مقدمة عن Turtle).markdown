##### **سكريبت الشرح (بلهجة مصرية) - 60 دقيقة**
**0-5 دقايق: مقدمة ممتعة**
"يا شباب، جاهزين تبقوا فنانين؟ النهاردة هنتعلم إزاي نرسم أشكال حلوة زي مربعات ودواير باستخدام حاجة اسمها **Turtle**! تخيلوا إنكم بتدّوا أوامر لسلحفاة صغيرة تمشي على الشاشة وترسم بالقلم. هنعمل رسومات ممتعة تخليكم تحسوا إنكم في معرض فني. يلا بينا!"
**5-20 دقيقة: شرح Turtle الأساسي**
"Turtle هي مكتبة في بايثون بتخليك ترسم أشكال بسهولة. عشان نستخدمها، لازم نكتب `import turtle` في أول الكود. خلونا نجرب نرسم خط بسيط:
</div>

<div dir="ltr">

```python
import turtle
turtle.forward(100)  # تمشي 100 خطوة
turtle.done()
```

</div>

<div dir="rtl">

لما نشغل الكود، هتظهر نافذة والسلحفاة هترسم خط مستقيم طوله 100 بكسل.
خلونا نفهم الكود:
- `import turtle` يعني جيب مكتبة Turtle عشان نستخدمها.
- `turtle.forward(100)` يعني خلي السلحفاة تمشي لقدام 100 خطوة وترسم خط.
- `turtle.done()` يعني خلّص الرسم وخلي النافذة مفتوحة.
نقدر نعمل حاجات تانية زي:
- `turtle.right(90)`: لف يمين 90 درجة (زي زاوية قايمة).
- `turtle.left(90)`: لف شمال 90 درجة.
- `turtle.penup()`: ارفع القلم (يعني امشي من غير ما ترسم).
- `turtle.pendown()`: انزل القلم (يعني ارسم تاني).
خلونا نجرب نرسم مربع:
</div>

<div dir="ltr">

```python
import turtle
turtle.forward(100)  # خط 1
turtle.right(90)     # لف يمين
turtle.forward(100)  # خط 2
turtle.right(90)     # لف يمين
turtle.forward(100)  # خط 3
turtle.right(90)     # لف يمين
turtle.forward(100)  # خط 4
turtle.done()
```

</div>

<div dir="rtl">

هيرسم مربع لأننا مشينا 100 خطوة 4 مرات، ولفينا 90 درجة بعد كل خط.

##### **الهدف التعليمي**
- فهم أساسيات مكتبة Turtle للرسم.
- استخدام أوامر `forward`, `right`, `left`, `penup`, `pendown`.
- رسم أشكال بسيطة مثل خطوط ومربعات.
##### **مثال عملي**
</div>

<div dir="ltr">

```python
import turtle
# رسم خط
turtle.forward(100)
turtle.done()
# رسم مربع
turtle.forward(100)
turtle.right(90)
turtle.forward(100)
turtle.right(90)
turtle.forward(100)
turtle.right(90)
turtle.forward(100)
turtle.done()
```

</div>
