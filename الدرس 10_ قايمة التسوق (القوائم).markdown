##### **سكريبت الشرح (بلهجة مصرية) - 60 دقيقة**
**0-5 دقايق: مقدمة ممتعة**
"يا شباب، تخيلوا إنكم بتعملوا قايمة تسوق للسوبر ماركت، زي: لبن، جبنة، شيكولاتة. في البرمجة، نقدر نعمل قايمة زي دي ونحط فيها حاجات كتير، ونستخدمها بسهولة! النهاردة هنتعلم حاجة اسمها **القوائم**، وهنخلي الكمبيوتر يعرض القايمة أو يضيف حاجات جديدة. جاهزين؟ يلا!"
**5-20 دقيقة: شرح القوائم**
"في بايثون، القايمة (List) زي صندوق كبير بتحط فيه حاجات كتير، زي أسماء، أرقام، أو حتى كلام. بنعمل قايمة باستخدام الأقواس المربعة `[]`. خلونا نجرب مثال:
```python
shopping_list = ['milk', 'cheese', 'chocolate']
print(shopping_list)
```
لما نشغل الكود، هيطلع: ['milk', 'cheese', 'chocolate'].
خلونا نفهم الكود:
- `shopping_list` هو اسم الصندوق الكبير (القايمة).
- `['milk', 'cheese', 'chocolate']` دي الحاجات جو القايمة، وكل حاجة بين علامات اقتباس عشان دي نصوص.
- الفواصل `,` بتفصل بين الحاجات.
- `print(shopping_list)` بيطبع القايمة كلها.
نقدر نطبع حاجة واحدة من القايمة باستخدام رقم مكانها (اسمه Index). العد بيبدأ من 0، يعني:
- `milk` في المكان 0.
- `cheese` في المكان 1.
- `chocolate` في المكان 2.
مثلاً:
```python
shopping_list = ['milk', 'cheese', 'chocolate']
print(shopping_list[1])
```
هيطلع: cheese، عشان `1` يعني ثاني حاجة في القايمة.
نقدر نضيف حاجة للقايمة باستخدام `append`. مثلاً:
```python
shopping_list = ['milk', 'cheese', 'chocolate']
shopping_list.append('juice')
print(shopping_list)
```
هيطلع: ['milk', 'cheese', 'chocolate', 'juice'].
نقدر كمان نكرر القايمة باستخدام حلقة `for`. مثلاً:
```python
shopping_list = ['milk', 'cheese', 'chocolate']
for item in shopping_list:
    print('I need to buy', item)
```
هيطلع:
I need to buy milk
I need to buy cheese
I need to buy chocolate
*(ارسم على السبورة قايمة زي دفتر مكتوب فيه 'milk، cheese، chocolate'، ووضح إزاي `append` بيضيف حاجة جديدة.)*

##### **الهدف التعليمي**
- فهم القوائم وكيفية إنشاؤها باستخدام `[]`.
- الوصول لعناصر القايمة باستخدام الـIndex.
- إضافة عناصر باستخدام `append` وتكرار القايمة بـ`for`.
##### **مثال عملي**
```python
# إنشاء قايمة
shopping_list = ['milk', 'cheese', 'chocolate']
print(shopping_list)
# طباعة عنصر واحد
print(shopping_list[1])
# إضافة عنصر
shopping_list.append('juice')
print(shopping_list)
# تكرار القايمة
for item in shopping_list:
    print('I need to buy', item)
```
##### **النشاط التطبيقي**
- **المهمة 1**: كتابة قايمة تسوق بـ3 حاجات وطباعتها.
- **المهمة 2**: إضافة حاجة جديدة وطباعة القايمة باستخدام `for`.
- **الكود**: كما في المثال العملي.
- **الطريقة**: كتابة الكود في Thonny أو Replit.
##### **التحدي الصغير**
- **السؤال**: اكتب كود يطبع رسالة لكل اسم في قايمة بـ4 أسماء.
- **مثال إجابة**:
  ```python
  friends = ['John', 'Sarah', 'Mike', 'Emma']
  for name in friends:
      print('Hello', name, ', how are you?')
  ```
##### **نصايح للمعلم**
- وضح فكرة القايمة بمثال زي دفتر درجات المدرسة.
- لو طفل كتب مفتاح غلط (زي `'milk'` بدل `'Milk'`)، اشرح إن المفتاح لازم يكون زي ما كتبناه بالظبط.
- شجعهم يجربوا قوايم مضحكة، زي قايمة أبطال خارقين ومهاراتهم.
- لو في وقت، خليهم يجربوا حذف مفتاح باستخدام `pop()` (بس اشرحها الأول).
- لو في أخطاء، زي نسيان `: ` بين المفتاح والقيمة، اشرح إنها زي رابط بين الاسم والمعلومة.
