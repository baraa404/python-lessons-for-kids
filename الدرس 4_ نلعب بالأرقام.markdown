##### **سكريبت الشرح (بلهجة مصرية) - 60 دقيقة**
**0-5 دقايق: مقدمة ممتعة**
"يا شباب، جاهزين نلعب بالأرقام زي الرياضيات، بس بطريقة ممتعة؟ النهاردة هنتعلم إزاي نستخدم الأرقام في البرمجة، زي إنك بتحسب فلوسك أو درجاتك في الامتحان. هنخلي الكمبيوتر يجمع، يطرح، يضرب، ويقسم بدل ما نستخدم الآلة الحاسبة. يلا، هنعمل حاجات جامدة!"
**5-20 دقيقة: شرح العمليات الحسابية**
"في بايثون، نقدر نستخدم الأرقام في حاجات كتير. الأرقام نوعين:
- **أعداد صحيحة** زي 5 أو 10، يعني من غير كسور.
- **أعداد عشرية** زي 5.5 أو 2.3، يعني فيها كسور.
نقدر نعمل عمليات زي دي:
- الجمع `+`، زي 3 + 2 = 5.
- الطرح `-`، زي 5 - 2 = 3.
- الضرب `*`، زي 3 * 2 = 6.
- القسمة `/`، زي 6 / 2 = 3.
خلونا نجرب نستخدم الأرقام في متغيرات. مثلاً، لو عندي 3 تفاحات و2 برتقال، نقدر نحسب المجموع:
```python
تفاح = 3
برتقال = 2
المجموع = تفاح + برتقال
print("عندي", المجموع, "فاكهة")
```
لما نشغل الكود، هيطلع: عندي 5 فاكهة.
خلونا نفهم الكود:
- `تفاح` و`برتقال` صناديق فيها أرقام (3 و2).
- `المجموع` صندوق فيه ناتج الجمع (3 + 2 = 5).
- في `print`، بنطبع كلام `"عندي"` و`"فاكهة"` بين علامات اقتباس، وصندوق `المجموع` من غير علامات.
- الفواصل `,` بتفصل بين الكلام والصناديق.
نقدر نجرب عملية تانية، زي الضرب. لو عايز أحسب تكلفة 3 حلوى، وكل حلوى بـ2 جنيه:
```python
حلوى = 3
سعر_واحد = 2
تكلفة = حلوى * سعر_واحد
print("التكلفة", تكلفة, "جنيه")
```
هيطلع: التكلفة 6 جنيه.
لو عايز أستخدم قسمة، مثلاً أحسب نص 10 جنيه:
```python
فلوس = 10
نص_الفلوس = فلوس / 2
print("نص الفلوس:", نص_الفلوس)
```
هيطلع: نص الفلوس: 5.0.
خلونا نفهم حاجة مهمة: لو قسمنا 10 على 2، النتيجة 5.0 (رقم عشري)، عشان بايثون بيحب يطلع النتيجة بدقة. لو عايزين نعرف باقي القسمة، بنستخدم `%`. مثلاً:
```python
عدد = 7
باقي = عدد % 2
print("باقي القسمة:", باقي)
```
هيطلع: باقي القسمة: 1، عشان 7 ÷ 2 = 3 والباقي 1.
*(ارسم على السبورة مثال زي: '3 تفاحات + 2 برتقال = 5'، ووضح إزاي الكمبيوتر بيحسب.)*
**20-40 دقيقة: نشاط تفاعلي**
"هنعمل نشاط ممتع! كل واحد هيعمل برنامج يحسب تكلفة شراء 4 حلوى، لو كل حلوى بـ3 جنيه. اكتبوا الكود كده:
```python
حلوى = 4
سعر_واحد = 3
تكلفة = حلوى * سعر_واحد
print("تكلفة الحلوى:", تكلفة, "جنيه")
```
افتحوا Thonny أو Replit، اكتبوا الكود، واضغطوا Run. لو ظهر 'تكلفة الحلوى: 12 جنيه'، يبقى نجحتوا!"
*(امنح 10 دقايق يكتبوا ويجربوا. ساعدهم لو نسيوا علامة `*` أو كتبوا الكود غلط.)*
"لعبة صغيرة: غيروا عدد الحلوى لـ5 أو 6، وحسبوا التكلفة من جديد. جربوا كمان عملية طرح، زي لو عندك 10 جنيه واشتريت حلوى بـ6 جنيه، كام الباقي؟ اكتبوا الكود وشوفوا!"
*(امنح 5 دقايق إضافية للتجربة، وشجعهم يشاركوا النتايج.)*
**40-55 دقيقة: تحدي صغير + مناقشة**
"عندنا تحدي! كل واحد هيعمل برنامج يحسب متوسط 3 درجات في المدرسة. يعني لو عندك درجات زي 80، 90، 70، هتحطهم في صناديق، تجمعهم، وتقسمهم على 3. اكتبوا الكود كده:
```python
درجة1 = 80
درجة2 = 90
درجة3 = 70
المجموع = درجة1 + درجة2 + درجة3
المتوسط = المجموع / 3
print("متوسط درجاتي:", المتوسط)
```
جربوا، ولو ظهر 'متوسط درجاتي: 80.0'، يبقى صح! مين هيشارك درجاته؟"
*(امنح 5 دقايق يكتبوا، وبعدين اطلب من 2-3 أطفال يقولوا النتيجة.)*
**55-60 دقيقة: تلخيص وتشجيع**
"برافو عليكم! النهاردة اتعلمنا إزاي نستخدم الأرقام في البرمجة، ونحسب زي الجمع والضرب والقسمة. المرة الجاية هنتعلم إزاي نكتب كلام زي رسايل ونضمهم مع بعض. فكروا في حاجات عايزين تحسبوها، زي فلوس أو درجات، وهنشوفها مع بعض!"
##### **الهدف التعليمي**
- فهم أنواع الأرقام (صحيحة وعشرية).
- استخدام العمليات الحسابية (`+`, `-`, `*`, `/`, `%`).
- كتابة برامج بسيطة للحساب باستخدام متغيرات.
##### **مثال عملي**
```python
# جمع أرقام
تفاح = 3
برتقال = 2
المجموع = تفاح + برتقال
print("عندي", المجموع, "فاكهة")
# ضرب أرقام
حلوى = 3
سعر_واحد = 2
تكلفة = حلوى * سعر_واحد
print("التكلفة", تكلفة, "جنيه")
```
##### **النشاط التطبيقي**
- **المهمة**: كتابة برنامج يحسب تكلفة 4 حلوى (كل حلوى بـ3 جنيه)، وتجربة تغيير العدد أو عملية الطرح.
- **الكود**:
  ```python
  حلوى = 4
  سعر_واحد = 3
  تكلفة = حلوى * سعر_واحد
  print("تكلفة الحلوى:", تكلفة, "جنيه")
  ```
- **الطريقة**: كتابة الكود في Thonny أو Replit.
##### **التحدي الصغير**
- **السؤال**: اكتب كود يحسب متوسط 3 درجات.
- **مثال إجابة**:
  ```python
  درجة1 = 80
  درجة2 = 90
  درجة3 = 70
  المجموع = درجة1 + درجة2 + درجة3
  المتوسط = المجموع / 3
  print("متوسط درجاتي:", المتوسط)
  ```
##### **نصايح للمعلم**
- وضح العمليات بأمثلة من الحياة، زي شراء حاجات أو توزيع فلوس.
- لو طفل كتب علامة غلط (زي `+` بدل `*`)، اشرح الفرق بمثال.
- شجعهم يجربوا أرقام مختلفة أو عمليات زي `%`.
- لو في وقت، خليهم يحسبوا حاجة خيالية، زي 'تكلفة 100 حلوى'.
- لو في أخطاء، زي القسمة على صفر، اشرح إن الكمبيوتر بيطلع خطأ ويحتاج نغير الرقم.
