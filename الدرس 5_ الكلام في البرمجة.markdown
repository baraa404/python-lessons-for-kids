"في بايثون، الكلام زي الأسماء أو الجمل بيكون بين علامات اقتباس `""` أو `''`. دول اسمهم **نصوص** أو Strings. مثلاً:
</div>

<div dir="ltr">

```python

```

</div>

<div dir="ltr">


</div>

<div dir="rtl">

هيطلع: Ali.
نقدر نضم نصوص مع بعض باستخدام علامة `+`. مثلاً، لو عايز أدمج اسم أول وأخير:
</div>

<div dir="ltr">

```python

last_name = "Mohamed"
full_name = first_name + " " + last_name

```

</div>

<div dir="rtl">

لما نشغل الكود، هيطلع: My name is Sara Mohamed.
خلونا نفهم الكود:
- `first_name` و`last_name` صناديق فيها نصوص.
- `" "` دي مسافة بين علامات اقتباس، عشان يبقى فيه فراغ بين 'Sara' و'Mohamed'.
- لو نسيت المسافة، هيطلع: SaraMohamed، وده مش حلو!
- علامة `+` هنا بتلزق النصوص مع بعض، زي إنك بتكتب كلمتين ورا بعض.
- في `print`، بنطبع كلام `"My name is"` وصندوق `full_name`.
نقدر نضم نصوص بطريقة تانية، زي إنك تكتب جملة كاملة. مثلاً:
</div>

<div dir="ltr">

```python
greeting = "Hello"
name = "Ahmed"
message = greeting + " " + name + "!"
print(message)
```

</div>

<div dir="rtl">

##### **الهدف التعليمي**
- فهم النصوص (strings) وكيفية تخزينها في متغيرات.
- دمج النصوص باستخدام `+`.
- كتابة برامج بسيطة لإنشاء رسائل.
##### **مثال عملي**
</div>

<div dir="ltr">

```python
# دمج نصوص
first_name = "Sara"
last_name = "Mohamed"
full_name = first_name + " " + last_name
print("My name is", full_name)
# عمل رسالة
greeting = "Hello"
name = "Ahmed"
message = greeting + " " + name + "!"
print(message)
```

</div>

<div dir="rtl">

##### **النشاط التطبيقي**
- **المهمة**: كتابة برنامج يدمج اسم أول وأخير، ثم كتابة برنامج لتحية صديق.
- **الكود**: كما في المثال العملي.
- **الطريقة**: كتابة الكود .
##### **التحدي الصغير**
- **السؤال**: اكتب كود يعمل رسالة باستخدام 3 نصوص (تحية، اسم، هواية).
- **مثال إجابة**:
  </div>

<div dir="ltr">

```python
  greeting = "Hello"
  name = "Sara"
  hobby = "drawing"
  message = greeting + " " + name + ", I love " + hobby + "!"
  print(message)
  ```

</div>

##### **نصايح للمعلم**
- وضح فكرة دمج النصوص بمثال زي كتابة رسالة واتساب.
- لو طفل نسي المسافة `" "`، اشرح إنها بتفصل الكلام زي المسطرة.
- شجعهم يكتبوا رسايل مضحكة، زي 'Hello Ahmed, I love food!'
- لو في وقت، خليهم يجربوا يضموا 4 نصوص أو أكتر.
- لو في أخطاء، زي كتابة نص من غير علامات اقتباس، اشرح الفرق بين نص ومتغير.
